# RAG Assignment

## Overview
This is a Retrieval-Augmented Generation (RAG) system built using Streamlit.

# RAG Q&A System

## Features
- Upload PDF or TXT files
- Ask questions from the uploaded document
- Uses Cohere embeddings
- Streamlit interface
